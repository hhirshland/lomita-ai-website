# Lomita AI — Website

Simple Next.js + Tailwind landing page using the **Meridian** brand direction from the Lomita AI brand system.

## Brand reference

- `docs/brand-guidelines.md` — written companion
- `public/lomita-brand-system.json` (root project) — color + type tokens
- Meridian direction: dark navy background, Source Serif 4 display, JetBrains Mono labels, muted peach accent

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
